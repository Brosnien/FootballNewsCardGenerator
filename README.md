# FootballNewsCardGenerator

Single-file generator for football news / transfer / result cards.

**Live app:** https://brosnien.github.io/FootballNewsCardGenerator/generator-ios.html

## How it works
`generator-ios.html` is fully self-contained (all CSS, JS, and images inline).
Edit it, commit, and push to `main` — GitHub Pages redeploys the live app
automatically within about a minute.

On iPhone: open the live link in Safari → Share → **Add to Home Screen** to run
it full-screen like an app.
