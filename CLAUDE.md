# Football News Card Generator — working notes

A browser app that generates football news / transfer / result cards, served from GitHub
Pages. It is **not** a single HTML file: page structure, styling, logic, team data and fonts
are separate cached static files. [README.md](README.md) has the file table, the teams.json
format, and the crest-backdrop notes — read it before assuming a layout.

## Plan of record

[ROADMAP.md](ROADMAP.md) holds every planned feature and tweak, in the author's own wording,
with done/not-done status.

**After any prompt that modifies this repo, update `ROADMAP.md` in the same commit:**

- tick items that the change completed, and add a short note saying where they landed;
- add any newly agreed item to the right section;
- refresh the `_Last updated:_` line with the date and a one-line summary of what changed.

Don't reword or reorder the author's items, and ask before removing one.
